# Changelog

## v1.1.0

发布时间：2026-03-07

### 新增
- 支持二级分类（一级/二级文件夹）管理。
- 查看页每条回复支持一键跳转编辑。
- 管理页新增多条件搜索（标题、内容、标签、分类）。
- 管理页新增批量操作：批量改分类、批量加标签、批量删标签、批量删除、批量复制副本。
- 新增数据导出/导入（JSON）。
- 导入支持策略选择：仅预览 / 合并导入 / 覆盖导入，并提供导入统计。

### 优化
- 主页面与管理页面侧栏均支持拖拽调整宽度，并持久化保存。
- 分类操作按钮改为按需显示（悬停/激活/展开），减少默认占位。
- 侧栏样式统一（整列高度、背景一致），提升视觉一致性。

### 拖拽能力增强
- 分类支持同级拖拽排序。
- 分类支持层级拖拽（拖入一级分类成为二级分类）。
- 管理页回复支持拖拽排序。
- 管理页回复支持拖拽到分类进行快速迁移。

### 修复
- 修复回复无法拖拽到二级分类的问题。
- 修复管理页侧栏最小宽度限制过大、无法缩小的问题。
- 修复主/管页面若干交互逻辑不一致的问题。

### 兼容性
- 兼容 v1.0.0 旧数据结构，支持平滑迁移到二级分类体系。
